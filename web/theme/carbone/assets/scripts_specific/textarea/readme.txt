counter(length, text)

counter permet d'ajouter une limite de caractère sur une textarea. Un champ texte est ajouté sous la textarea pour indiquer le nombre de caractères actuel.

	length est le nombre limite de caractères autorisés pour la textarea.
	text est une chaîne de caractères qui sera affichée après l'élement texte du compteur.

L'argument text est facultatif.
